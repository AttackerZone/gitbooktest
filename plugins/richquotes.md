# richquotes

https://plugins.gitbook.com/plugin/richquotes


After installation Be sure too activate the option from the book.json file :

```
{
    "plugins"       : ["richquotes"]
    ,"pluginsConfig":
    {
        "richquotes" :
        {
            "todos" : true /*false by default*/
        }
    }
}
```
